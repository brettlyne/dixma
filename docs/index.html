Installing the Dixma Plugin

The plugin should be available in the Figma community soon. Until then, you can install manually by following these instructions.

This method only works for the Figma desktop client.

1. Download Plugin Files

  1. Go to https://github.com/brettlyne/dixma
  2. Click the green "Code" button
  3. Click the last option: “Download Zip”
  4. Unzip the downloaded file

2. Install the Plugin

In the Figma menu go to:

  • Plugins >
  • Development >
  • New Plugin

Choose “Click to choose a manifest.json file”

Select the manifest.json file in the folder you just unzipped

3. Launch the Plugin

In the Figma menu go to:

  • Plugins >
  • Development >
  • Dixma
